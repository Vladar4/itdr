Printing Instructions

  • Page setup:
    • normal booklets: Two-sided - short edge (flip)
    • reversed booklets: Two-sided - long edge (standard)
  • Page scaling: none
  • Auto rotate and center: yes
  • Trim printed booklets to about 5.4in x 8in.
